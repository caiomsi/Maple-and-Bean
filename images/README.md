# Images — Maple & Bean

This is an **agency demo site**. It ships with real, warm coffee-shop photos
sourced from [Unsplash](https://unsplash.com) (free for commercial use, no
attribution required) so the demo looks finished out of the box. The `<img>` tags
still keep a `placehold.co` `onerror` fallback as a safety net.

**Before launching for a real client:** replace these stock photos with the
client's own / properly licensed photography, keeping the same
**descriptive, hyphenated, lowercase filenames** below (they're an SEO signal and
the `alt` text references them). Drop a new file over the old one and you're done.

| Filename (recommended)   | Where it's used              | Recommended size       | What to show |
|--------------------------|------------------------------|------------------------|--------------|
| `hero-coffee-pour.jpg`   | **Hero background**          | 1920×1080 (landscape)  | Warm, inviting shot — a coffee pour, the bar, or the cafe at golden hour |
| `our-roastery.jpg`       | Our Story section            | 700×800 (portrait)     | Owner/barista at the roaster or behind the bar |
| `latte-art.jpg`          | Gallery slot 1               | 600×600 (square)       | Latte art in a ceramic cup |
| `cafe-interior.jpg`      | Gallery slot 2               | 600×600 (square)       | The interior — wood, plants, seating |
| `pastry-case.jpg`        | Gallery slot 3               | 600×600 (square)       | Croissants / pastries in the bakery case |
| `pour-over.jpg`          | Gallery slot 4               | 600×600 (square)       | A hand pour-over being brewed |
| `coffee-beans.jpg`       | Gallery slot 5               | 600×600 (square)       | Freshly roasted beans, close up |
| `window-seat.jpg`        | Gallery slot 6               | 600×600 (square)       | Cozy window seat with a coffee |
| `storefront.jpg`         | Schema `image` (SEO)         | 1200×800               | Storefront / signage / branded shot |
| `social-share.jpg`       | Open Graph share image       | 1200×630               | Branded image shown when the link is shared |

## Tips
- Compress images before committing (TinyPNG / Squoosh) — big photos slow the site.
- JPG for photos, PNG only for logos/graphics with transparency.
- Keep gallery shots roughly square so the 1:1 grid crops cleanly.
